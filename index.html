Develop, deploy, and edit websites
like never before

Enough with writing code in lifeless files and editing content in dreary forms (we love you WordPress but come on). primo lets you jump right into the code, edit content on the frontend, and publish a crazy fast, anywhere-hosted static site.
Oh, and did we mention it's free and open-source?

Get to the code

Edit your site component-by-component (each with encapsulated styles) and use Tailwind out of the box alongside your favorite JS libraries with zero setup.

Edit with ease

Editing on the frontend isn't just for page-builders anymore. Write and edit text right on the page and add fields in seconds to edit component content.

Create a component library

Create a new component, add it to your library, then place it anywhere on your site. And when you update it, every instance of it across your site gets updated too.

Build a static site

Everytime you publish your changes, primo builds a lightning-fast static site (with just HTML and CSS, plus a couple enhancements) which you can deploy to your host of choice.
